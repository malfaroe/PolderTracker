# PolderTracker

![Build Debug APK](https://github.com/malfaroe/PolderTracker/actions/workflows/build.yml/badge.svg)

Tracker personal de sesiones de Grounding. App nativa Android (Kotlin + Jetpack Compose).

## Compilar

Haz push a `main` → GitHub Actions compila el APK → descárgalo desde la pestaña **Actions** del repositorio.
